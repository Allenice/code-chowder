计算公式

Available space = (container size - flex-basis siblings total)
Grow unit = (Available space / Sum Grow siblings total).
Flex item size = (Flex basis + (Grow Unit * num))

flex: grow shrink basis

  • 2/0/100px
  • 0/0/150px
  • 2/0/100px
Available space = 600 - (100 + 150 + 100) = 250
Grow unit = Available space / (2 + 0 + 2) = 62.5

change basis

  • 2/0/100px
  • 0/0/150px
  • 2/0/50px
Available space = 600 - (100 + 150 + 50) = 300
Grow unit = Available space / (2 + 0 + 2) = 75

shrink

  • 1/1/200px
  • 1/2/200px
  • 1/3/200px
  • 1/4/200px
Available size = 600 - (200*4) = -200
shrink unit = available size / (1+2+3+4) = -20
item width = basis - (shrink num * shrink unit)
当 Available size 是负数时， item 的宽度与 grow 的值无关
